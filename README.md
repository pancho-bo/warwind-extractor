# Extract sprites from War Wind game data files

* For now it outputs sprite sheets in the format readily usable by [Stratagus](https://github.com/Wargus/stratagus) game engine.
* If you want to browse the list of sprites in the datafile you can use [retroGDX](https://github.com/IceReaper/retroGDX)

## Configs
What to extract and where to output can be configured in scripts.lua file.

## Usage
zig build run
